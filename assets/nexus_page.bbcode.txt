[size=4][b]Description[/b][/size]
[color=#D4D4D8]DLL version[/color] of the Map for Goblins mod. Thousands of loot and world icons on the in-game map, and [b]no regulation.bin changes[/b]: all map data gets injected into memory at runtime, so it won't conflict with other mods that edit the regulation.

There are three builds on the Files tab, pick the one for your game:
[list]
[*][b]ERR version[/b], for [url=https://www.nexusmods.com/eldenring/mods/541]ERR[/url]. ~9000 icons, including the ERR-specific stuff (Rune Pieces, Ember Pieces, Kindling Spirits, unique loot etc).[/*]
[*][b]Vanilla version[/b], for the base game + Shadow of the Erdtree. ~6700 icons, same mod, just without the ERR-only categories.[/*]
[*][b]Convergence version[/b], for [url=https://www.nexusmods.com/eldenring/mods/3419]The Convergence[/url]. ~7200 icons generated from the overhaul's own data, so it covers its relocated loot, new bosses and reworked zones. Matches the Convergence version it was built against (currently 2.2.3A) - after a Convergence update expect some markers to be off until this build is updated too.[/*]
[/list]
Based on [url=https://www.nexusmods.com/eldenring/mods/3091]Map for Goblins[/url] by Harmonixer and [url=https://github.com/Gacsam/Goblin-ERR]Goblin-ERR[/url] by Gacsam, rebuilt from scratch as a single DLL + gfx for icons.

[size=4][b]Main features[/b][/size]
[list]
[*]Icon categories that can be turned on/off individually in [b]MapForGoblins.ini[/b]: weapons, armour, talismans, spells, key items, smithing stones, gloveworts, cookbooks, bell bearings, gathering nodes, bosses, graces, stakes of Marika, spirit springs, imp statues, paintings, summoning pools, map fragments, gestures, NPC invaders, puzzles etc.[/*]
[*]Icons in a region only show up after you've found that region's map fragment (can be disabled).[/*]
[*]Things you've already collected disappear from the map: including gathered material nodes (like Arteria Leaf), Rune/Ember Pieces etc. Defeated bosses either hide or get a checkmark (configurable).[/*]
[*]Every marker shows its location name, down to sub-areas (like Siofra Aqueduct/Siofra River etc). The mod uses original game strings, everything should be localised out of the box.[/*]
[*]The INI maintains itself. If there's no file - it gets created. If there are new options after an update - they will be added automatically.[/*]
[*]F10 (or Y+R3 on gamepad) hides/shows all the icons on the fly.[/*]
[*]The ERR version can optionally tie ERR's own pre-placed markers (camps, merchants, bosses, dungeon entrances) to the same display rules as this mod's icons, so everything on the map follows one logic: show after map discovery, hide once cleared. If you prefer ERR's markers as they are, turn these options off in the ini.[/*]
[/list]

[size=4][b]Installation - ERR version[/b][/size]
[list=1]
[*]Download the ERR file and unpack it[/*]
[*]Copy [b]dll/[/b] and [b]addons/[/b] into your ERR folder (merge with what's there)[/*]
[*]Launch through ERR as usual[/*]
[/list]
Settings are in [b]dll/offline/MapForGoblins.ini[/b]

[size=4][b]Installation - Vanilla version[/b][/size]
You need a mod loader, either [b]ModEngine3 (me3)[/b] or [b]ModEngine2[/b].

[b]Option A: ModEngine3 (me3), recommended[/b]
[list=1]
[*]Install me3 ([url=https://github.com/garyttierney/me3/releases]me3_installer.exe[/url], docs at [url=https://me3.help/]me3.help[/url])[/*]
[*]Copy the [b]MapForGoblins[/b] folder from the archive next to your .me3 profile (e.g. eldenring-default.me3)[/*]
[*]Open the .me3 profile in Notepad and make sure it has:
[code]profileVersion = "v1"

[[supports]]
game = "eldenring"

[[packages]]
path = 'MapForGoblins'

[[natives]]
path = 'MapForGoblins/MapForGoblins.dll'[/code]
[/*]
[*]Double-click the .me3 file to play[/*]
[/list]
[b]Option B: ModEngine2[/b]
[list=1]
[*]Get [url=https://github.com/soulsmods/ModEngine2/releases]ModEngine2[/url] and extract it anywhere (NOT into the game folder)[/*]
[*]Copy the [b]MapForGoblins[/b] folder into the ModEngine2 directory (next to launchmod_eldenring.bat)[/*]
[*]In [b]config_eldenring.toml[/b] add the DLL and the asset folder:
[code]external_dlls = [ "MapForGoblins\\MapForGoblins.dll" ]

mods = [
    { enabled = true, name = "default", path = "mod" },
    { enabled = true, name = "MapForGoblins", path = "MapForGoblins" }
][/code]
[/*]
[*]Run launchmod_eldenring.bat[/*]
[/list]
Settings are in [b]MapForGoblins/MapForGoblins.ini[/b]

[size=4][b]Installation - Convergence version[/b][/size]
[list=1]
[*]Download the Convergence file and unpack it[/*]
[*]Copy the [b]MapForGoblins[/b] folder into your ConvergenceER folder (next to Start_Convergence.bat)[/*]
[*]In [b]config_eldenring.toml[/b] add the DLL to external_dlls and the asset folder ABOVE the "mod" entry (first entry wins file conflicts - our world-map file already contains the Convergence map UI):
[code]external_dlls = [
    ...existing entries...,
    "MapForGoblins\\MapForGoblins.dll",
]

mods = [
    { enabled = true, name = "MapForGoblins", path = "MapForGoblins" },
    { enabled = true, name = "mod", path = "mod" }
][/code]
[/*]
[*]Launch through Start_Convergence.bat as usual[/*]
[/list]
Settings are in [b]MapForGoblins/MapForGoblins.ini[/b]. Note: the Convergence Launcher may rewrite config_eldenring.toml when it updates the mod - re-check the two edits after updates.

[size=4][b]Requirements[/b][/size]
[list]
[*]ERR version: [url=https://www.nexusmods.com/eldenring/mods/541]ERR[/url] itself, nothing else. The mod loads through ERR's own loader.[/*]
[*]Vanilla version: [url=https://github.com/garyttierney/me3/releases]ModEngine3[/url] or [url=https://github.com/soulsmods/ModEngine2/releases]ModEngine2[/url][/*]
[*]Convergence version: an installed [url=https://www.nexusmods.com/eldenring/mods/3419]The Convergence[/url] (it ships its own ModEngine2). Shadow of the Erdtree is required by the overhaul itself.[/*]
[*][color=#ff0000][b]OFFLINE ONLY.[/b][/color] Mod loaders disable EAC, so the "Inappropriate activity detected" message at launch is normal. Don't try to play online with mods. If you host Seamless Co-op sessions and something goes wrong, you may temporarily toggle off icons by pressing F10.[/*]
[/list]
